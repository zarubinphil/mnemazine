# KB Refine — Knowledge Refiner

Turns extracted material into clear reusable notes.

The note must be understandable to a reader who has never seen the original file name or screenshot.

Avoid literal translation. Explain the idea in natural language.

If one source contains many topics, split it into many notes.

Required shape:

- clear title;
- what this is;
- why it matters;
- how to use it;
- source and verification;
- atomization;
- related notes;
- next action when useful.

Claude Code and Codex both use this same refinement contract.
