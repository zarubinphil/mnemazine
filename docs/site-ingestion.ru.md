# Парсинг сайтов

🇷🇺 **Русский** · [🇬🇧 English](site-ingestion.md)

Запуск:

```bash
node scripts/mnemazine-ingest-site.mjs --url https://example.com --apply --graphify --max-pages 40
```

Парсер находит страницы через:

- `robots.txt`;
- ссылки sitemap;
- `sitemap.xml`;
- ссылки того же домена на стартовой странице.

Для каждой страницы извлекает:

- заголовок;
- подзаголовки;
- видимый текст;
- ссылки на GitHub;
- URL источника.

На выходе — markdown-заметка со статусом проверки. Заметка не считается идеальным финальным знанием, пока её не отсмотрят и при необходимости не разобьют.

Авторизация намеренно исключена из парсера по умолчанию. Публичные open-source пользователи не должны утекать сессии браузера, cookie или приватные дашборды в конвейер знаний.
