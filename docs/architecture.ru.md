# Архитектура

🇷🇺 **Русский** · [🇬🇧 English](architecture.md)

У Mnemazine пять слоёв.

## 1. Inbox

Сырые файлы попадают через `inbox/`.

Примеры:

- скриншоты;
- PDF;
- выгруженные веб-страницы;
- markdown-заметки;
- транскрипты;
- ссылки, сохранённые как `.md`-файлы.

## 2. Извлечение

Извлечение — local-first:

- прямое чтение текста;
- MarkItDown;
- Apple Vision OCR;
- Whisper;
- кастомные парсеры.

Извлечение — это ещё не знание.

## 3. Огранка

Агент или workflow решает:

- о чём источник;
- содержит он одну тему или много;
- какие утверждения требуют внешней проверки;
- какие заметки уже есть и должны быть обновлены.

## 4. Vault

Vault содержит готовое знание. Это Obsidian-совместимый markdown.

## 5. Граф и отчёты

Graphify строит карты связей. Еженедельные HTML-отчёты делают память читаемой и применимой.
