# Экономика токенов

🇷🇺 **Русский** · [🇬🇧 English](token-economics.md)

Mnemazine сокращает расход токенов, удерживая повторяющуюся, механическую и уже известную работу вне контекста модели.

## Локальная работа стоит ноль модельных токенов

Эти операции проходят без вызовов LLM:

- перечисление файлов;
- хэширование;
- обнаружение дублей;
- OCR;
- извлечение из PDF;
- транскрипция;
- обновление графа;
- проверки gate качества.

## Hash-кэш

Каждый файл inbox получает SHA-256 хэш. Если тот же файл появляется снова, Mnemazine может его пропустить.

## Атомизация

Длинный гайд часто содержит много несвязанных идей. Хранение его одной гигантской заметкой заставляет будущие промпты читать слишком много. Разбиение на атомы позволяет агенту извлекать только нужную часть.

## Извлечение через Graphify

Graphify помогает извлекать связанный контекст через структуру графа. Это дешевле, чем класть весь vault в промпт.

## Финальные заметки

Финальные заметки меньше сырого OCR, потому что убирают:

- дублирующийся текст;
- навигацию;
- шаблонный boilerplate;
- битый OCR;
- нерелевантные скриншоты;
- повторяющиеся вступления.

Результат — vault, запросы к которому со временем дешевеют.
